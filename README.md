# FOREST PLANNING AND MANAGEMENT IN SANGGAU REGENCY (WEST KALIMANTAN) AND KUTAI KARTANEGARA REGENCY (EAST KALIMANTAN)

## CONTEXT
Baseline assessment of forest cover, High Conservation Value (HCV) areas,
policy frameworks, and stakeholder networks in Other Land Use Areas (APL).

## MY ROLE
- GIS analysis and map production
- Biodiversity data collection
- Field survey data on socio economic
- Drone piloting
- Report writing and visualization

## DATA AND METHOD
- Government land-use and forest status maps
- Field survey data (HCV, socio-economic)
- Spatial analysis
- Drone operation
- Stakeholder mapping and SWOT analysis
  **Tools:** ArcGIS Pro, QGIS, field survey methods, drone mapping

## KEY OUTPUTS
- HCV distribution maps (HCV 1–6)
- Forest cover status in APL
- Stakeholder centrality analysis
- Policy and funding recommendations
  
## DATA AND REPRODUCIBILITY
Due to data-sharing agreements and project confidentiality, the full datasets
and detailed workflows are not publicly available. The repository presents
selected outputs and methodological summaries for demonstration purposes.
